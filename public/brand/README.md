# Assets de marque YM Studio

Ce dossier accueillera les futurs assets de marque officiels. Aucun logo n'est
créé pour cette première version : l'interface et le PDF utilisent un
monogramme temporaire (`YM`) généré en CSS.

À déposer ici lorsqu'ils seront disponibles :

- `logo-primary.svg` — logo principal (fond clair)
- `logo-dark.svg` — variante du logo pour fonds sombres
- `logo-pdf.svg` — logo destiné à la couverture des propositions PDF
- `favicon.svg` — favicon du site
